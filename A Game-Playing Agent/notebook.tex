    \subsection{Title: ** Mastering the game of Go with deep neural networks
and tree search
**}\label{title-mastering-the-game-of-go-with-deep-neural-networks-and-tree-search}

\subsubsection{Personal Reason to choose
AlphaGod:}\label{personal-reason-to-choose-alphagod}

\textbf{I have chose AlphaGo by DeepMind Team because it is about game
of Go and a computer program never defeated a huma professional player
in a full size game of Go.}

\subsubsection{Introduction:}\label{introduction}

\textbf{The game of Go has long been viewed as the most challenging of
classic games for artificial intelligence owing to its enormous search
space and the difficulty of evaluating board positions and moves. Deep
Mind team introduced a new approach to computer Go that uses `value
networks' to evaluate board positions and `policy networks' to select
moves.}

    \subsubsection{Game of Go Difficulties:}\label{game-of-go-difficulties}

\textbf{infeasible exhaustive search}: In large games, such as chess
(b≈35, d≈80) and especially Go (b≈250, d≈150), exhaustive search is
infeasible.

    \subsubsection{DeepMind Approach:}\label{deepmind-approach}

\textbf{DeepMind introduce a new approach} to computer Go that uses
\textbf{`value networks'} to evaluate board positions and
\textbf{`policy networks'} to select moves. These deep neural networks
are trained by a novel combination of supervised learning from human
expert games, and \textbf{reinforcement learning} from games of
self-play.

Deep Mind also introduce a new search algorithm that combines Monte
Carlo simulation with value and policy networks.

    \subsubsection{Methods used by Deep
Mind}\label{methods-used-by-deep-mind}

 1. \textbf{Monte Carlo tree search} : estimate the value of each state
in a search tree. As more simulations are executed, the search tree
grows larger and the relevant values become more accurate. The policy
used to select actions during search is also improved over time, by
selecting children with higher values.

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\setcounter{enumi}{1}
\item
  \textbf{policy network}: Used Convolutional Neural Network to read the
  board as an image and finds the positions on the board, then using
  supervised learning from human moves to learn the game following by
  Reinforcement learning to improve the network.
\item
  \textbf{value network}: evaluate board position.
\end{enumerate}

    \subsubsection{Results:}\label{results}

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  ** By combining tree search with policy and value networks, AlphaGo
  has finally reached a professional level in Go, providing hope that
  human-level performance can now be achieved in other seemingly
  intractable artificial intelligence domains **
\item
  ** AlphaGo achieved a 99.8\% winning rate against other Go programs,
  and defeated the human European Go champion by 5 games to 0. This is
  the first time that a computer program has defeated a human
  professional player in the full-sized game of Go, a feat previously
  thought to be at least a decade away**
\end{enumerate}
